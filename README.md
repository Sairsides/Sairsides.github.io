# SairSides Portfolio (Astro)

Personal linktree/portfolio site for **https://sairsides.github.io**.

## Run locally
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
npm run preview
```

## Deploy
Deployment is automatic via GitHub Actions on push to `main`.

## Edit content
Main editable content is in:

- `src/data/profile.ts` (bio, social links, certificates, badges, iframe profile)

## Theme/style
- Dark glassmorphism with dark-red accents
- Responsive, mobile-first
- Motion respects `prefers-reduced-motion`
